lint-lang implementation notes

Goal

Lint grammar locally for free with a Rust CLI backed by local GGUF models.

Review notes

  • Moved CLI argument definitions to src/cli.rs so domain command parsing stays isolated.
  • Moved default lint execution to src/lint.rs.
  • Added domain commands: lint, models list, models download, and models select.
  • Added lint-lang doctor for setup diagnostics without entering the TUI.
  • Kept model files out of git. GGUF and llamafile files are large and user-specific.
  • Kept README focused on installation, first run, examples, and development commands.
  • Added README links for Rust, llama.cpp, llama-cpp-rs, GGUF, Qwen3, and Metal.
  • Kept common Rust commands in the Makefile.
  • Kept native llama.cpp on macOS with Metal for the default local path.
